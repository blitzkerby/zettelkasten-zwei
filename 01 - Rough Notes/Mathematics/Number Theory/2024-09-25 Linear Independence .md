A set of functions:
$$\large{
	\{ y_i(x) \}^{n}_{i=1} 
	= 
	\{ y_1(x), \ ... \ , y_n(x)\}
}$$ 
are considered to be linearly independent if and only if:
$$\large{
	{c_1}{y_1(x)} + {...} + {c_n}{y_n(x)} = 0
}$$
implies:
$$\large{
	{c_1} = {...} = {c_n} = 0
}$$
